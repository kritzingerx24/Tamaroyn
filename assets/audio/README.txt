
TO USE AUTHENTIC WULFRAM 2 SOUNDS:
Replace the files in this folder with the .ogg files from your repository:

engine_loop.wav -> tank.ogg (or internal_tank.ogg)
shoot.wav       -> gunshot4.ogg
explosion.wav   -> explo1.ogg
jump.wav        -> tank_jump3.ogg
